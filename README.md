# Diabetes Risk Prediction POC

Educational and research proof of concept for predicting diabetes risk using
machine learning and storing results in a database.

## Project Goal
This project aims to build a prototype application that:
- accepts relevant medical input data
- generates a diabetes risk prediction using a machine learning model
- stores the results in a database
- allows later review and analysis

## Project Structure
- `backend/` - API and machine learning logic
- `frontend/` - user interface
- `data/` - datasets
- `models/` - trained/exported models
- `docs/` - notes and planning
- `scripts/` - helper scripts

## Status
Initial local setup completed.

## Disclaimer
This project is for educational and research purposes only and does not provide
medical diagnosis.
